## About

OSIRIS는 저수준 공개 IR을 축으로 컴파일러·런타임·서빙·벤치마크를
아우르는 오픈소스 AI 시스템SW 생태계 구축에 기여하는 것을 지향합니다.

### 비전

저수준 공개 IR을 중심으로 AI 시스템 SW 오픈 생태계에 참여하고,
지속가능한 기여·협력 체계를 마련하는 것을 목표로 합니다.
